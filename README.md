# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/kunal-r0209/MyPortfolio
```

## 2️⃣ Navigate into Project

```bash
cd Kunal-Portfolio-React
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Run Development Server

```bash
npm run dev
```
